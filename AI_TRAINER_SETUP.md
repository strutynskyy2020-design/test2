# AI-тренажер: налаштування Netlify

1. Завантажте зміни в GitHub. Netlify автоматично запустить новий deploy.
2. У Netlify відкрийте **Project configuration → Environment variables**.
3. Створіть секретну змінну `OPENAI_API_KEY`.
4. У полі **Production** вставте ключ `sk-proj-...`.
5. Для Scope достатньо **Functions**. Якщо ваш тариф не дозволяє вибір scope, залиште доступні стандартні scope.
6. Необов'язково створіть `OPENAI_MODEL` зі значенням `gpt-4.1-mini`.
7. Запустіть **Deploys → Trigger deploy → Deploy site**.

Після входу на сайт у нижньому меню з'явиться вкладка **AI**. Ключ ніколи не передається у браузер: він читається лише Netlify Function.

## Голосове введення на мобільних

Версія використовує запис мікрофона через MediaRecorder і серверне розпізнавання OpenAI. Це працює у сучасних Safari на iPhone/iPad, Chrome на Android, Chrome/Edge/Safari на комп'ютері. Максимальна тривалість одного запису — 30 секунд.

На iPhone сайт має бути відкритий через HTTPS (Netlify це забезпечує). При першому натисканні кнопки мікрофона дозвольте доступ до мікрофона. Для встановлення як застосунку: Safari → Поділитися → На початковий екран.

Для транскрипції за замовчуванням використовується `gpt-4o-mini-transcribe`. За потреби модель можна змінити змінною Netlify `OPENAI_TRANSCRIBE_MODEL`.


## Голосове введення без витрат OpenAI
Голос перетворюється на текст через браузерний Web Speech API. Запити до OpenAI Audio API не виконуються. Якщо браузер не підтримує SpeechRecognition, використайте кнопку диктування на системній клавіатурі iPhone або Android.
