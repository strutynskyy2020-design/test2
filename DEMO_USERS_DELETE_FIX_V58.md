# V58 — видалення демо-користувачів

Причина: `Анна Коваль`, `Максим Дубенко` та `Олена Ткач` були записані в `SEED_DEMO_USERS` і створювалися повторно під час кожного запуску backend.

У v58 демо-користувачі за замовчуванням не створюються. Для окремого демонстраційного середовища їх можна повернути змінною:

```env
SEED_DEMO_USERS=true
```

У production цю змінну не додавайте або встановіть `false`. Після оновлення backend видаліть трьох користувачів через адмін-панель ще раз. Вони більше не повернуться після перезапуску.
