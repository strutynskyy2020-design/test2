# TM6 Bonus v66 — Bonus Match: анімації, спецфішки та рівні-виклики

## Frontend

Файл: `frontend/src/pages/BonusMatch.jsx`

Додано:

- Framer Motion для вибору, свапу, shake невалідного ходу, зникнення збігів і падіння фішок.
- Стабільні `id` клітинок і `layoutId`, тому фішки візуально переміщуються між клітинками.
- Хвильове зникнення збігів, міні-конфеті в точці збігу та політ очок до лічильника.
- Відображення `КОМБО ×2`, `КОМБО ×3` тощо.
- Анімації активації ракети, бомби та джокера.
- Перемога: послідовна поява зірок і `fireConfetti()`.
- Програш: shake і затемнення поля.
- Екран-попередження перед бос-рівнями 25, 40 і 50.
- Підтримка `prefers-reduced-motion` для слабких пристроїв і користувачів зі зменшеними анімаціями.

## Backend

Файл: `backend/server.py`

Клітинка дошки тепер має структуру:

```json
{
  "id": "stable-id",
  "symbol": "coin",
  "special": null,
  "obstacle": null,
  "obstacle_hits": 0
}
```

Спецфішки:

- `rocket_row` — очищує рядок;
- `rocket_col` — очищує колонку;
- `bomb` — очищує область 3×3;
- `color_bomb` — прибирає всі фішки типу, з яким її поміняли.

Створення:

- 4 у ряд → ракета;
- 5 у ряд → джокер;
- Т/Г-подібний збіг → бомба.

Сервер повертає покрокову анімаційну інформацію: дошку після свапу, клітинки збігу, створені й активовані спецфішки, дошку після очищення, дошку після падіння та нові фішки.

## Прогресивна складність

Кожен 5-й рівень:

- має стрибок цільового рахунку;
- дає менше ходів;
- вводить нову перешкоду;
- показується як рівень-виклик.

Перешкоди відкриваються послідовно:

`ice → chain → crate → stone → crystal → web → shield → slime → metal → core`

Рівні 25, 40 і 50 є бос-рівнями та мають збільшений множник XP/Point до застосування денного ліміту Point.

## Перевірки

- `python -m py_compile backend/server.py` — успішно.
- JSX-синтаксис перевірено через TypeScript compiler з `--allowJs --jsx react-jsx --noEmit` — успішно.
- Окремо перевірено генерацію дошок для рівнів 1, 5, 10, 25, 40, 50, наявність можливого ходу, відсутність стартових збігів, створення джокера/бомби, падіння навколо нерухомих перешкод і пошкодження перешкод.
