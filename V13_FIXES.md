# V13 точкові виправлення

- Фото профілю: окрема видима кнопка «Змінити фото» та прямий нативний file picker.
- Підтримка JPEG, PNG, WebP, GIF, HEIC і HEIF, навіть якщо браузер не передав MIME-тип.
- Адмінка: ширина до 1500 px задається безпосередньо в AppLayout, тому не залежить від Tailwind media-класів.
- На екранах до 1023 px зберігається мобільний картковий вигляд.
- Архів має плоску структуру репозиторію, без додаткової папки v12work.
