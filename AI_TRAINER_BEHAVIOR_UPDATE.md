# Оновлення логіки поведінки клієнта

- Прибрано завершення кейсу лише через 8 реплік.
- Терпіння і готовність тепер можуть як зростати, так і знижуватися залежно від змісту та тону відповіді.
- Нецензурна лексика, прямі образи й погрози завершують кейс одразу зі скаргою без додаткового API-запиту.
- Тиск, крик великими літерами та неправдиві абсолютні гарантії дають жорсткий штраф.
- Пороги успіху: 70% для легкого, 80% для середнього, 90% для важкого кейсу.
- Початкове терпіння: 90 / 72 / 52 відповідно.
