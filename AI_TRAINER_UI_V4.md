# AI Trainer UI v4

Оновлення:
- мобільний чат використовує окрему прокручувану область, поле відповіді завжди залишається видимим;
- компактніші повідомлення, аватар і ім'я клієнта;
- окрема картка оцінювання з розкривним блоком «Приклад сильної відповіді»;
- етап розмови, лояльність і ймовірність угоди у верхній панелі;
- анімований індикатор набору відповіді клієнтом;
- фінал показує окремо якість консультації та ймовірність продажу;
- голосове введення до 60 секунд через Web Speech API без OpenAI-транскрипції;
- у Safari, де Web Speech API недоступний або нестабільний, залишається підказка використовувати мікрофон системної клавіатури.

Після оновлення файлів зробіть Commit, Push і новий Deploy у Netlify.
