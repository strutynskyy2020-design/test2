# V57: нульові видачі для відсутнього оператора

Якщо оператор відсутній у блоці `giving month` або `giving yesterday` на вкладці `Transformation Deb`, застосунок показує 0 видач у всіх напрямках замість порожнього екрана.

Оновлено два рівні:

- `Code.gs` повертає нульовий рядок для наявного періоду, де логін не знайдений.
- frontend заповнює відсутній період нулями під час поширення нової версії Apps Script або кешування.

Після заміни `integrations/google-sheets/Code.gs` створіть нову версію deployment у Google Apps Script.
